Comparing test artifacts with existing JIRAs

GemFire JIRA: https://jira-pivotal.atlassian.net/

After you have downloaded and extracted the artifacts, first we try to see that if there are tickets already opened with the same failure.

  1. Search for the stacktrace in JIRA
  1. Read through the JIRAs
  1. Try to match the symptoms mentioned in the JIRA with the downloaded artifacts for the failure you are trying to analyze.
  1. If all symptoms match convincingly, mark the failure in HydraDB.
